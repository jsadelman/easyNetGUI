Welcome to easyNet

    easyNet is a computational modeling software package for cognitive science.

    Our aims in developing the package are to:

  • Improve communication about models among cognitive scientists.
  • Allow programming and modeling novices greater access to the practice of modeling.
  • Facilitate more complete integration of modeling and experimentation.
  • Expand modeling practice to include more thorough testing and comparison of models.
