Specifying the output

Before we actually run the trial, let’s specify what sort of output we’d like to see. For this example, we are particularly interested in what effect this letter string will have at the word level. In order to monitor activities at that level, let’s create a plot. Right click anywhere on the word level and choose the pop-up menu option that says Set up default plot.

Now we’re ready to run the trial. Click the green run button in the control bar. You may notice that the message changes briefly to a message while the model is running, and then returns to displaying when it is finished. You've just run your first trial!
